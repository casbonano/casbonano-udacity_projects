HTML Markup =

- < > (opening tag)
- "content"
- < / > (closing tag>)

Some tags include:

b = bold tag
a = anchor tag
em = emphasis tag or italics

There are also other tags, such as img, for adding images

HTML Attributes
< tag attr="value"> content
- tags can have multiple Attributes
- anchors are for making links

Example:

< img src="url" alt="text"

-alt text required (as in, complaints if not there).

-images are void tags. images do not need a closing tag because they do not have content.


White Space

br tag. Also a void tag. stands for "break".

p = paragraph.
p is NOT a void tag. This tag has content. The content = the actual paragraph body.


Span and Div

<span> = inline
<div> = block

These two tags contain content and allow styles to be attached to them.

<span class="foo" >text< /span>
<div class="bar" >text< /div>
